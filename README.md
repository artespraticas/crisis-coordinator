# 🌐 Global Coordination Agent — Crisis Coordinator

> **The Synthesis Hackathon 2026** — Built by Reis Mesquita

An AI-powered disaster response dashboard demonstrating how autonomous agents can solve civilization-scale coordination failures. The agent integrates real-time data, reasons about resource allocation, and issues directives — all logged permanently on Base Mainnet.

## 🚀 Live Demo

👉 https://crisis-coordinator-zeta.vercel.app

## ⚡ What It Does

1. **Simulate a crisis** — Click ⚡ to trigger a war strike: 12 city nodes, cascading failures, resource conflicts
2. **Activate the AI** — Click ◈ to call Claude live with streaming real-time reasoning
3. **Watch coordination** — Directives fire in sync with map animations, resources reallocate
4. **On-chain logging** — Every session logged to Base Mainnet via smart contract

- **Deployment:** Vercel

## 🌍 The Vision

Most unsolvable problems — climate change, housing crises, disaster response — are coordination failures, not resource failures. The Global Coordination Agent is a prototype OS for solving these at civilizational scale.
