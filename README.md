# 🌐 Global Coordination Agent — Crisis Coordinator

> **The Synthesis Hackathon 2026** — Built by Reis Mesquita (Claudio) + Claude

An AI-powered disaster response dashboard demonstrating how autonomous agents solve civilization-scale coordination failures in real time.

## 🚀 Live Demo

👉 **[https://crisis-coordinator-zeta.vercel.app/](#)** 

## ⚡ How It Works

1. Click **⚡ Strike Event** — triggers a war strike on a city grid of 12 nodes
2. Click **◈ Activate Agent** — Claude Sonnet activates and streams live reasoning
3. Watch directives fire in sync with map animations across hospitals, rescue teams, depots and shelters
4. Every session is logged permanently to Base Mainnet

## 🌍 The Vision

Most of humanity's hardest problems — climate change, housing, disaster response — are coordination failures, not resource failures. The Global Coordination Agent is a prototype OS for solving these at civilizational scale.
